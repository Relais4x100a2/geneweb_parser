Outils pour la propsographie - Parser de fichier Geneweb (*.gw)

Ne perdez plus de temps à recopier les arbres geneanet, utilisez notre parser ! Nous sommes très heureux de vous proposer le premier outil libre de Toldot. Son code est disponible sous licence MIT via le dépôt github. Entièrement gratuit, vous pouvez nous soutenir en adhérant à Toldot.

Modèle de données proposées

Pour rappel, le modèle Geneweb : page de documentation officielle

Notre proposition de modèle :

  • Individu
  • Organisation (une famille étant considérée comme une organisation)
  • Evenement

Evolutions à venir

Cette première version est ammenée à évoluer en fonction de vos retours. N'hésitez donc pas nous faire part de vos observations et de vos suggestions ! Soit via le dépôt Github, soit par mail.

Nous nous efforcerons dans un premier temps d'améliorer le code et la modélisation proposée.

Nous pensons proposer à terme une possibilité pour les utilisateurs de laisser une copie de leurs données sur notre serveur afin de pouvoir les avertir en cas de correspondances avec les données d'un autre utilisateur.

Que deviennent vos données ?

Vos données sont automatiquement effacées à l'issue de votre requête sur notre serveur.

A moyen terme, nous envisageons de proposer aux utilisateurs de laisser une copie de leurs données sur notre serveur et de les avertir en cas d'éventuelles correspondances avec les données d'un autre utilisateur.
