Ne perdez plus de temps à recopier les arbres geneanet, utilisez notre parser ! Nous sommes très heureux de vous proposer le premier outil libre de Toldot. Son code est disponible sous licence GNU GPL via le dépôt github. Entièrement gratuit, vous pouvez nous soutenir via notre page sur HelloAsso.

Modèle de données proposé

Pour rappel, le modèle Geneweb : page de documentation officielle

Notre proposition de modèle :

  • Individu
  • Organisation (une famille étant considérée comme une organisation)
  • Événement
Nous expliquerons plus en détail notre démarche dans la prochaine version du site.

Évolutions à venir

Cette première version est amenée à évoluer en fonction de vos retours. N'hésitez donc pas à nous faire part de vos observations et de vos suggestions ! Soit via le dépôt Github, soit par mail : dev/strudel/toldot/point/fr

Nous nous efforcerons dans un premier temps d'améliorer le code, la documentation et la modélisation proposée.

Nous pensons proposer à terme une possibilité pour les utilisateurs de laisser une copie de leurs données sur notre serveur afin de pouvoir les avertir en cas de correspondances avec les données d'un autre utilisateur.

Que deviennent vos données ?

Vos données sont automatiquement effacées à l'issue de votre requête sur notre serveur. A moyen terme, nous envisageons de proposer aux utilisateurs de laisser une copie de leurs données sur notre serveur et de les avertir en cas d'éventuelles correspondances avec les données d'un autre utilisateur.
